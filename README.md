# DL_Mini_Projects

A collection of mini Deep Learning projects implemented in Jupyter Notebooks.

## 📁 Projects

### 1. CNN.ipynb
**Convolutional Neural Network (CNN)**

- Image classification using Convolutional Neural Networks
- Built with TensorFlow/Keras (or PyTorch)
- Typical use case: Computer Vision tasks

### 2. Iris_prediction_ANN.ipynb
**Artificial Neural Network (ANN) for Iris Dataset**

- Multi-class classification on the classic Iris flower dataset
- Fully connected neural network implementation
- Great introductory project for understanding neural network basics, backpropagation, and classification

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- TensorFlow / Keras (or PyTorch)
- NumPy, Pandas, Matplotlib/Seaborn
